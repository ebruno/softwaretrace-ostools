◆ swtrprcmgt_set_maxpid()

int swtrprcmgt_set_maxpid ( SWTPROC_MGT *  ctrl)

Get max PID allowed by the system. The max pid is stored in /proc/sys/kernel/pid_max This function reads the value and stores in max_pid in the control block. if an error occurs the value SWTRPOCMGT_DEFAULT_MAX_PID is set.

For FreeBSD sysctl is used it can also be used on Linux to obtain the max pid value but we use the proc file system under Linux for consistency.

Parameters
ctrlPointer Control Block for the library.
Returns
0 or -1 if an error occurs.

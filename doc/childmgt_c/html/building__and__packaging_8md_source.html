1 ## Building and Packaging ##
2 Supported Operating Systems are show below:
3  * ArchLinux 4.N
4  * CentOS 7.N
5  * Fedora 27.N
6  * FreeBSD 11.N (FreeBSD version 9.N should also work)
7  * RedHat Enterprise Linux 7.N
8  * Ubuntu 16.N
9 
10 Cmake version 2.8 is the mininum version supported.
11 The cmake will create build environment for the c and c++ versions of the tools.

122 ### Packaging ###
123 To build a package use the make file Makefile_packages
124 cmake will be run as part of package creation process.
125 
126  make -f Makefile_packages <rule name>
127  For example if specify the ArchLinux rule on CentOS the following will be displayed:
128  [user@system c_cpp]$ make -f Makefile_packages swtrstrlib_create_distro
129 
130  Checking to see if OS/Distro ""CentOS Linux"" is supported
131  OS/Distro ""CentOS Linux"" is supported, continuing.
132  make: *** No rule to make target `swtrstrlib_create_distro'. Stop.
133 
134 Note rules only work on stated operating system.
135 
136 | Operation System | packaging rule name |
137 |------------------------|---------------------------|
138 | RHEL,CentOS, Fedora | swtrstrlib_create_rpm, swtrprocmgtutilslib_create_rpm |
139 | Ubuntu | swtrstrlib_create_pkg, swtrprocmgtutilslib_create_pkg |
140 | ArchLinux | swtrstrlib_create_distro, swtrprocmgtutils_create_distro |
141 | FreeBSD | \todo Provide packaging for FreeBSD
